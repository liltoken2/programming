Дан файл с текстом в формате XML, в котором используются только теги и числовые значения, причем у тэгу не может быть и вложенного в него тэгу, и числового значения одновременно.
Каждая строка гарантированно заканчивается знаком ">". Имена вложенных тегов не повторяются. Имена тегов - последовательность латинских букв.
Нужно эту систему тегов из формата xml переписать в другой файл в формате json. Использование библиотек для работы с xml и json запрещено
 
In.txt:
 
<a><bb>
<c>1</c>
<d>3</d></bb>
</a>
 
Out.txt:
 
{"a":{"bb":{"c":1, "d":3}}, "median":2}
