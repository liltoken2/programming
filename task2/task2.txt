дан файл с текстом в формате XML, в котором используются только тэги и числовые значения, причем у тэга не может быть и вложенного в него тэга, и числового значения одновременно. 
Каждая строка гарантированно заканчивается знаком ">". Имена вложенных тэгов не повторяются. Имена тэгов - последовательность латинских букв. 
Нужно эту систему тэгов из формата xml переписать в другой файл в формате json. Использование библиотек для работы с xml и json запрещено
 
In.txt:
 
<a><bb>
<c>1</c>
<d>3</d></bb>
</a>
 
Out.txt:
 
{"a":{"bb":{"c":1, "d":3}}, " median":2}
 
еще надо добавить в конец  медиану всех числовых значений.
