В файле in.txt записаны арифметические выражения, состоящие из операции сложения и вычитания над натуральным числами.
Каждое выражение начинается с новой строки и не содержит пробелов.

Требуется:
1. Переписать в файл out.txt эти выражения, добавив к ним результат их вычисления после знака =

2. В файл out2.txt записать исходные выражения, указав через пробел после каждого из них, какое минимальное количество вычитаемых нужно удалить, чтобы результат стал не отрицательным. Строки в файле должны располагаться в порядке увеличения значений выражения из файла out.txt

Файл В ПАМЯТЬ ЦЕЛИКОМ ЧИТАТЬ НЕЛЬЗЯ, РАБОТАТЬ ПОСТРОЧНО.

Использовать функцию eval запрещено.

Пример in.txt:

2+3
4-1+2-10

Out.txt
2+3=5
4-1+2-10=-5
